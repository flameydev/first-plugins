# What are Plugins?

Plugins are tools used to speed up the workflow for Roblox Developers. They can be as simple as adding a part, to as complex as simulating 2D particles with physics.

Plugins can be installed from the Toolbox or Creator Store. Once installed, they will appear in the Plugins Tab to be used.

Plugins can be really handy, as they manage to do some things, that are normally very time consuming, instantly.
